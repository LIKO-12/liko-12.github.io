LIKO-12

An open-source fantasy computer

Get Started

Create retro-looking games

For a fantasy system with a low resolution of 192x128 and limited to a colors palette of 16 colors.

Share your games with others

By sending them an image looks like a disk, but also magically contains your game.

Available on multiple platforms

Developed on Windows and supports Linux. Experimental support for Android and macOS is also available.

Code in Lua

An easy to learn programming language.

Open-source

You can look into the code that makes it work and even hack it to do new things!
